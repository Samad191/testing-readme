*  --from, --fromBlock <fromBlock>               From Block

*  -h, hash                                      Get tx hash
*  --m, merge                                    Merge files
*  --u, unique                                   Get unique hashes
*  --r, repeating tx hashes                      Get repeating hashes
*  --help                                        Display help for command
